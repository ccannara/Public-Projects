Grid Based Example
Made by Cameron Cannara

This project is an example of a grid based movement system video game. This was programmed in C 
using gbmb18 and bgtd22, both of which can be found here: http://www.devrs.com/gb/hmgd/
These files are meant to be compiled using make.bat, resulting in "main.gb". This file is a 
ROM that can ONLY be run using a Gameboy Emulator or real Gameboy hardware using a third-party cartridge.
Please be advised, this ROM is NOT to be sold as a file or on a catridge. This is not an official game.
