Python Stock Market Assistant

Functionality:

> Check Specific Stock
  - Displays all possible stock data for a given ticker
  - Live Updates to current price
  - Utilize Machine Learning to predict the future price of a stock
  - Calculate Simple Moving Averages
  - See Company Balance Sheets, Cash Flow Statements, and Dividends

> See Best and Worst Performing Stocks of the past 24 hours
  - Be able to use Machine Learning to check trends

> Check Live Crypto Prices

Dependancies:

requests_html
yahoo_fin
numpy
pandas
pandas_datareader
install sklearn
